Proceso de inscripción (Para todos los niveles)

  • Realizar el pago mediante deposito o transferencia electronica a la cuenta correspondiente.
  • Enviar el comprobante de pago con el nombre del estudiante.
  • Una vez confirmado el pago, se comparte el enlace de Google Drive para la descarga del material.

Curso de Excel - Nivel I

Frecuencia Horario
Lunes a Jueves 9:00PM a 10:00PM

El material consiste en cuatro archivos:

Material de apoyo

1. Atajos de teclado

  • Contiene los atajos más importantes dentro de Microsoft Excel.
  • Los atajos están divididos por categoria.
  • Contiene la combinación de teclas en idioma inglés y español.
  • Descripción de la funcionalidad del atajo.

2. Diccionario de funciones

  • Contiene las funciones incorporadas en Excel para sus diferentes versiones.
  • Las funciones están divididas por categoria.
  • Nombre de las funciones en idioma inglés y español.
  • Disponibilidad de las funciones para las diferentes versiones.

3. Videos de clase

  • Archivo Word el cual contiene el temario del nivel correspondiente.
  • Por cada tema se adjunta un enlace que reddirecciona a un video del tema en la nube.
  • No existe la necesidad de descargar los videos y se pueden visualizar cada vez que se requiera.
  • Los videos sirven para anticiparse a las clases o para recuperar alguna clase perdida.

Material de trabajo

4. Nivel 1

  • Archivos Excel personalizados con VBA el cual contiene una hoja de trabajo por cada tema.
  • Descripción de las diferentes herramientas y funciones.
  • Atajos de teclado utilizados en cada tema.
  • Ejemplos
  • Ejercicios practicos.
